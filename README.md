# Historical Dates Timeline

Интерактивный таймлайн с историческими датами и событиями.

- Вращающийся круг с 6 периодами
- Только активная точка - полноразмерный синий круг
- Слайдер событий для каждого периода
- Адаптивный дизайн для всех устройств
- Анимации вращения и переключения

### Вариант 1:

Откройте `dist/index.html` в браузере. Все уже собрано и готово к использованию!

Вариант 2: Запустить локальный сервер

# 1. Установите зависимости
npm install

2. Запустите локальный сервер
npm start
# или
npm run dev

# 3. Откройте в браузере
Проект доступен прямо в браузере:  
**[https://captive444.github.io/historical-dates-timeline/](https://captive444.github.io/historical-dates-timeline/)**
